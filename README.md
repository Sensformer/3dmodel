# 3D 车载摄像头安装模拟器

这是一个基于 Three.js 开发的 3D 车载摄像头安装与视角模拟工具。它可以帮助您直观地配置车模尺寸、调整摄像头安装位置及参数，并实时预览各摄像头的视野。

## 🚀 如何运行

由于项目使用了 ES Modules，直接打开 HTML 文件可能会遇到跨域问题。建议使用本地服务器运行：

### 方法 1：使用 Python (推荐)
在项目根目录下运行：
```bash
python3 -m http.server 8080
```
然后在浏览器中访问：`http://localhost:8080`

### 方法 2：使用 Node.js (serve)
如果您安装了 Node.js，可以运行：
```bash
npx serve .
```

## 🛠 界面参数说明

右上角的控制面板（GUI）分为以下几个核心部分：

### 0. 车模尺寸 (0. 车模尺寸)
*   **车长/车宽/车高 (cm)**: 调整车模的物理尺寸，模型会实时按比例缩放。
*   **车厢比例**: 调整上层车厢相对于底盘的长度比例。
*   **相机安装高度 (cm)**: 统一调整四个摄像头距离地面的垂直高度。
*   **相机嵌入深度 (cm)**: 调整相机相对于车体边缘的嵌入深度，正值表示向内嵌入。
*   **标尺显示范围 (m)**: 调整地面 X 轴（红）和 Z 轴（蓝）参考标尺的覆盖长度。
*   **显示距离标尺**: 勾选以开启或关闭地面及坐标原点标注。
*   **显示标定布**: 勾选以在车身四个角落显示 2m x 2m 的 5x5 棋盘格标定布（每个方格 30cm，四周留白 25cm）。

### 0.5 标定布位置 (0.5 标定布位置)
*   **前后偏移 (cm)**: 调整标定布距离车身前后边缘的偏移值。这个值是标定布上离车身最近的角点到车身边缘的距离。
*   **侧方偏移 (cm)**: 调整标定布距离车身侧边缘的偏移值。这个值是标定布上离车身最近的角点到车身边缘的距离。

### 1. 相机视距与 FOV (1. 相机视距与 FOV)
*   **统一视距 (m)**: 调整所有相机的最远可见距离。
*   **焦距 (mm)**: 相机镜头焦距（1.0mm - 10.0mm）。焦距越短，视野越宽，鱼眼效果越明显。
*   **统一水平视场角 (deg)**: 同时调整四个相机的水平视野范围（10° - 190°）。**这是控制鱼眼效果的关键参数！**
*   **统一垂直视场角 (deg)**: 同时调整四个相机的垂直视野范围。
*   **视场显示方式**: 选择相机视野的可视化方式：
    *   `Lines`: 显示相机辅助线（默认方式）
    *   `Box`: 显示实心绿色视锥体
    *   `Both`: 同时显示线条和视锥体
    *   `None`: 隐藏所有视场可视化
*   **视场显示距离 (m)**: 调整视场可视化（线条和视锥体）的显示范围，与相机实际视距独立。
*   **当前成像模型**: 自动根据水平视场角 (hFOV) 切换：
    *   `hFOV < 90°`: **Pinhole (针孔)** - 正常透视投影，无畸变
    *   `90° <= hFOV < 160°`: **Kannala-Brandt (鱼眼)** - 多项式鱼眼投影，有明显的桶形畸变
    *   `hFOV >= 160°`: **Equidistant (等距)** - 等距鱼眼投影，更强的鱼眼效果

### 2. 水平安装角 (2. 水平安装角)
*   **前相机 (deg)**: 调整前视相机的水平朝向，默认 0°（朝向正前方）。
*   **后相机 (deg)**: 调整后视相机的水平朝向，默认 180°（朝向正后方）。
*   **左相机 (deg)**: 调整左视相机的水平朝向，默认 90°（朝向正左侧）。
*   **右相机 (deg)**: 调整右视相机的水平朝向，默认 -90°（朝向正右侧）。

### 3. 垂直安装角 (3. 垂直安装角)
*   **前/后/左/右相机 (deg)**: 分别调整各相机的俯仰角度。正值抬头，负值低头（如下俯视路面）。默认 -30°。

### 4. 虚拟环境控制 (4. 虚拟环境控制)
*   **太阳高度角/方位角**: 模拟不同时间段的太阳位置，实时改变天空颜色和车身阴影。
*   **大气/散射参数**: 调整大气的浑浊度和散射效果，模拟晴天或雾天。
*   **曝光强度**: 调整场景的整体亮度。
*   *注：此文件夹默认收起，点击可展开*

### 视角切换
*   **Main**: 全局观察视角，支持鼠标旋转、缩放。
*   **Front/Back/Left/Right**: 切换到对应的车载摄像头实时画面，此时轨道控制会自动禁用。**切换到这些视角才能看到鱼眼效果！**

## 🐟 鱼眼效果使用指南

### 如何查看鱼眼效果
1. 点击右下角的 **"当前视角"** 下拉菜单
2. 选择 **Front/Back/Left/Right** 中的任意一个
3. 展开 **"1. 相机视距与 FOV"** 文件夹
4. 拖动 **"统一水平视场角 (deg)"** 滑块

### 鱼眼效果变化
*   **hFOV < 90°**: 正常透视（针孔相机）
*   **90° - 160°**: Kannala-Brandt 鱼眼，边缘开始出现桶形畸变
*   **> 160°**: 等距鱼眼，强烈的鱼眼效果，画面呈圆形

### 鱼眼成像特点
*   中心区域放大，边缘区域收缩
*   直线会变成曲线（桶形畸变）
*   视野越大，畸变效果越明显

## 🖱 交互操作
*   **旋转**: 鼠标左键点击并拖拽（仅 Main 视角）。
*   **缩放**: 鼠标滚轮。
*   **平移**: 鼠标右键点击并拖拽。
