# 3D 车载摄像头安装模拟器

这是一个基于 Three.js 开发的 3D 车载摄像头安装与视角模拟工具。它可以帮助您直观地配置车模尺寸、调整摄像头安装位置及参数，并实时预览各摄像头的视野。

## 🚀 如何运行

由于项目使用了 ES Modules，直接打开 HTML 文件可能会遇到跨域问题。建议使用本地服务器运行：

### 方法 1：使用 Python (推荐)
在项目根目录下运行：
```bash
python3 -m http.server 8000
```
然后在浏览器中访问：`http://localhost:8000`

### 方法 2：使用 Node.js (serve)
如果您安装了 Node.js，可以运行：
```bash
npx serve .
```

## 🛠 界面参数说明

右上角的控制面板（GUI）分为以下几个核心部分：

### 0. 车模尺寸 (0. 车模尺寸)
*   **车长/车宽/车高 (cm)**: 调整车模的物理尺寸，模型会实时按比例缩放。
*   **相机安装高度 (cm)**: 统一调整四个摄像头距离地面的垂直高度。
*   **标尺显示范围 (m)**: 调整地面 X 轴（红）和 Z 轴（蓝）参考标尺的覆盖长度。
*   **显示距离标尺**: 勾选以开启或关闭地面及坐标原点标注。

### 1. 相机视距与 FOV (1. 相机视距与 FOV)
*   **统一视距 (m)**: 调整所有相机的最远可见距离（Far Plane），可通过相机助手的伸缩直观观察。
*   **前/后/左/右相机 (deg)**: 分别调整每个摄像头的视场角（广角程度）。

### 2. 水平安装角 (2. 水平安装角)
*   **Yaw (deg)**: 调整相机绕垂直轴的旋转。例如，前视相机默认 0°，左视相机默认 -90°。

### 3. 垂直安装角 (3. 垂直安装角)
*   **Pitch (deg)**: 调整相机的俯仰角度。正值抬头，负值低头（如下俯视路面）。

### 4. 虚拟环境控制 (4. 虚拟环境控制)
*   **太阳高度角/方位角**: 模拟不同时间段的太阳位置，实时改变天空颜色和车身阴影。
*   **大气/散射参数**: 调整大气的浑浊度和散射效果，模拟晴天或雾天。
*   **曝光强度**: 调整场景的整体亮度。

### 视角切换
*   **Main**: 全局观察视角，支持鼠标旋转、缩放。
*   **Front/Back/Left/Right**: 切换到对应的车载摄像头实时画面，此时轨道控制会自动禁用。

## 🖱 交互操作
*   **旋转**: 鼠标左键点击并拖拽（仅 Main 视角）。
*   **缩放**: 鼠标滚轮。
*   **平移**: 鼠标右键点击并拖拽。

## 📦 技术栈
*   [Three.js](https://threejs.org/) - 3D 渲染引擎
*   [lil-gui](https://github.com/georgealways/lil-gui) - 轻量级调试界面
*   Procedural Sky Shader - 程序化天空模拟
